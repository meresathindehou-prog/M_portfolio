SkillKit Hub

Plateforme e-commerce de produits digitaux adaptée au contexte africain

Problème

Vendre des produits digitaux (ebooks, guides, kits) dans 5 domaines différents avec un paiement adapté au marché béninois — sans Stripe ni PayPal.

Gestionnaire de Notes

Application C avec persistance fichiers, CRUD complet et statistiques

/* gestionnaire_notes.c */
float calculerMoyenne(float *t, int n) {
  float s = 0;
  for(int i=0; i<n; i++) s += t[i];
  return s / n;
}
Problème

Gérer et persister des notes étudiants en console, sans interface graphique.

Solution

Application C complète — pointeurs, tableaux dynamiques, persistance via fopen/fprintf/fscanf, menu interactif.

Résultats

~200 lignes couvrant CRUD, calculs statistiques (moyenne, max, min) et sauvegarde entre sessions.
